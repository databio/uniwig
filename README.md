# Uniwig

## usage:

```
uniwig bedfile stepsize smoothSize variableformat > file.wig 

smoothsize 0 - no smoothing
variableFormat 1 - for variable format, 0 - for fixed format

```


test command for chromosome print: 

```
./uniwig test.bed 1 5 0
```

## Makefile:

```
make uniwig
```

```
make test
```

```
make clean
```
